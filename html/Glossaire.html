Cybersécurité : C’est un moyen utilisé pour assurer la sécurité des systèmes et des données informatiques d’un État, d’une entreprise.

Cybercriminels : C’est un individu qui commet un ou plusieurs crime(s)à l’aide d’outils informatiques, comme par exemple : En piratant des données existantes sur Internet afin d’obtenir illégalement de l’argent ou un quelconque profit.

Pirater/Piratage : Le piratage est un acte malveillant qui est de s’introduire par effraction dans un réseau ou un ordinateur.

Windows : Windows est un système d’exploitation

Linux : Linux ou GNU/Linux est une famille de systèmes d’exploitation.

Ubuntu : Ubuntu est un système d’exploitation qui est basé sur Linux.

Gestionnaire de fichiers/FTP : C’est un logiciel utilisé dans le transfert de fichiers entre deux ordinateurs (FTP = File Transfert Protocol).

MOOC ANSSI : Le MOOC (Massive Open Online Course) est un outil de formation gratuit en ligne, accessible a tous qui à divers supports comme des vidéos, des exercices et des explications via des dessins.

HTML : C’est un type de langage informatique qui permet la mise en forme des pages Internet (HyperText Markup Language).

CSS : C’est un langage informatique permettant de mettre en forme des pages Internet (Cascading Style Sheets).

Sélecteurs : Un sélecteur CSS est un mot-clé qui permet de désigner une catégorie d’éléments.

JS/JQuery : JS qui est l’acronyme de JavaScript, qui est un langage informatique utilisé sur des pages Internet. JQuery est un framework de JavaScript qui permet de faciliter des fonctionnalités communes de JavaScript.

Framework : C’est un ensemble d’outils et de composants logiciels à la base d’un logiciel ou d’une application (infrastructure logicielle)

SLAM : SLAM (Solutions Logicielles et Applications Métier) est une formation qui est spécialisé dans le développement de site Internet.

SISR : SISR (Solutions d’Infrastructure, Systèmes et Réseaux) est une formation qui est spécialisé dans la gestion d’infrastructure réseau.

Infrastructure Réseau : Une infrastructure réseau est un ensemble d’équipements qui est composé des équipements physique (routeurs, switchs, ordinateurs, serveurs).
