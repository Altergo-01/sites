HTML5

Il s'agit d'un language de base pour la création de site Web, qui permet d'afficher un contenu (texte,images)
Créer des liens hypertext et relier les pages entre elles
De structurer un document (titre, paragraphe)
Indiquer au navigateur de ce qu'il doit afficher
Bien indexer votre page web dans les moteurs de recherches (choix de balise, mot-clé…)
Structure d'une page html :

Ligne 1 : type du document (voir obligatoire)
 Ligne 2 : balise html (idem)
 Ligne 3 : head> décrit notre page, sans que cette même chose soit affiché dans cette dernière, sans transcription visuelle, ce sont des instruction pour le navigateur, non pas pour l'utilisateur, c'est la qu'on trouvera les liens vers les page css…
 Ligne 4 : balise de type META (description) qui spécifie l'encodage de la page
 Ligne 5 : désigne comment s'affiche la page (initial scale = zoom initial)
 Ligne 6 : titre de la page, spécifie le titre de document
 Ligne 7 : balise fermante Head
 Ligne 8 : corps de la page, ce qui va être affiché.

CSS

Le CSS est un langage qui a été inventé pour styliser les contenus de nos pages en leur appliquant des styles.

pour styliser un notre contenu on doit faire appel à des "séléecteurs" comme id et class qui sont les plus vastements utilisé mais il est aussi possible de directement appeler la balise concerner, et faire donc prendre à toute celle présente dans notre HTML le style qu'on va lui donner, cependant une règle de priorité s'installe, ou pour la retenir, il faut se dire une chose

C'est toujours le style lié au selecteur le plus précis qui sera appliqué en priorité par exemple #p > .p > p

on applique des styles à l'aide de "propriété" comme background-color, color, font-family etc...

à ces propriétés on va appliquer des valeurs, comme le couleur de background-color, en anglais.

une propriété : valeur est appelé une déclaration.


il est possible de faire du style à l'intérieur d'un html avec la balise style, ou l'attribut style.

on commente dans un css avec /* chose à commenter */


ici d'autre spécificité d'un css :


* Sélectionne tous les éléments

E, F Sélectionne tous les éléments de type E et de type F

E F Sélectionne tous les éléments F à l’intérieur des éléments E

E > F Sélectionne les éléments F enfants directs des éléments E

E + F Sélectionne tout élément F placé directement après un élément E

E~F Sélectionne tout élément F placé après un élément E dans la page


!important à la fin d'une déclaration css fera passer cette dernière en priorité sur toute les autres
